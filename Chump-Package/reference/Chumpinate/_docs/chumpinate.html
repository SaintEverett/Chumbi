PackageVersion

inherits : Object

PackageVersion describes a specific version of your package: this includes all files, version number, etc.

examples

  • build-pkg-win.ck
  • build-pkg-mac.ck
  • build-pkg-linux.ck

constructors

PackageVersion()

Default constructor for PackageVersion.

PackageVersion(string pkg_name, string version)

Create a version of 'pkg_name', using a version string. This version string must follow the semantic versioning format major.minor.patch (i.e. "1.3.12")

PackageVersion(string pkg_name, int major, int minor, int patch)

Create a version of 'pkg_name', integers correspond to major.minor.patch semantic versioning.
