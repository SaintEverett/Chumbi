EncodeN

inherits : UGen

EncodeN is an ambisonic encoder meant for virtual source panning. Five editions exist, covering first to fifth order ambisonics. Declaring which order of ambisonics you want to use, replace "N" with the order, such as Encode1 for first order ambisonics.

SADN

inherits : UGen

Sampling Ambisonic Decoder. Follows same syntax as EncodeN, where SAD1 is a first order decoder. Best for optimal speaker layouts, where speakers are evenly spaced.

  • build-pkg-mac.ck
  • build-pkg-linux.ck
